# raylib-cs

C# bindings for raylib 2.0, a simple and easy-to-use library to learn videogames programming (www.raylib.com)

## Installation
Windows:
- Download the repository
- Open Raylib-cs.sln
- Copy x86 or x64 dll inside ExampleApplication to your output folder
- Build and run
